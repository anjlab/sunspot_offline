# Sunspot Offline
Simple failover method into your sunspot_rails + sidekiq setup

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'sunspot_offline'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install sunspot_offline
```
